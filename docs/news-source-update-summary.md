## ✅ 新しい6サイト構成

1. **MIT Technology Review** - アカデミック視点、深い技術分析
2. **VentureBeat AI** - ビジネス応用、投資動向
3. **TechCrunch AI** - スタートアップ、最新プロダクト
4. **The Verge AI** - 社会的影響、わかりやすい解説
5. **Towards Data Science** - ハンズオン、実践的技術記事
6. **Wired (AI)** - 深い調査報道、社会的影響、政策分析 ✨ **新規追加**

**重要**: 出力記事数は従来通り5-10件を維持（より厳選）

---

## 📊 6サイト構成にした理由

### 入れ替えではなく追加のメリット

1. **重複緩和**
   - より多くのソースから選択できるため、重複記事のリスクが減少
   - 特定サイトの更新が少ない日でも、他のサイトで補完可能
   
2. **品質の安定化**
   - 6サイトから厳選することで、より質の高い記事を選べる
   - 特定サイトの品質が低い日があっても影響を最小化

3. **多様性の向上**
   - Towards Data Science: 実践的な技術記事
   - Wired: ジャーナリズム・社会的影響
   - 両方の価値を活用できる

4. **柔軟性の確保**
   - 日によって各サイトからの記事数を調整可能
   - より戦略的な記事選定が可能

### Wired を追加した理由

1. **ジャーナリズムの質**
   - 30年以上の歴史を持つ権威あるテックメディア
   - 調査報道、深い分析、トレンド解説で定評
   - 編集基準が非常に高い

2. **社会的影響の深い分析**
   - AIの倫理、政策、文化的影響を深く掘り下げる
   - 技術と社会の橋渡し役として優れている

3. **読者層の拡大**
   - 技術者だけでなく、ビジネスリーダーや政策立案者にもアピール
   - 幅広い読者層に価値を提供

4. **他サイトとの補完性**
   - The Verge よりさらに深い社会的分析
   - MIT Technology Review と異なる角度からの技術解説

## 📈 期待される効果

### 1. 重複記事の削減
- 6サイトから厳選することで、より多様な記事を収集可能
- 同じトピックでも異なる視点の記事を選択できる

### 2. 記事品質の向上
- より多くの候補から選ぶため、質の高い記事を厳選可能
- 日によって各サイトの品質が変動しても、全体の品質を維持

### 3. カバレッジの拡大
- 技術、ビジネス、社会、実践の全側面をより確実にカバー
- 特定分野への偏りを防ぐ

### 4. バランスの改善

**6サイトの役割分担**:
- **MIT Technology Review**: 学術的深さ・研究動向
- **VentureBeat**: ビジネス応用・投資動向
- **TechCrunch**: スタートアップ・新プロダクト
- **The Verge**: 一般読者向け解説・消費者視点
- **Towards Data Science**: 実践的技術・ハンズオン
- **Wired**: 調査報道・社会的影響・政策分析

すべての重要な視点がバランスよくカバーされています。
